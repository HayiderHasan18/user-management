# User Management Project

This is a full-stack project with a **frontend** and **backend**, connected to an Aiven MySQL database.

## Project Structure

- `frontend/` - Frontend application (React / Vue / Angular)  
  See [frontend/README.md](frontend/README.md) for setup instructions.

- `backend/` - Backend API (Node.js / Express)  
  See [backend/README.md](backend/README.md) for setup instructions.

## Features

- User registration & login
- CRUD operations on users
- MySQL database via Aiven
